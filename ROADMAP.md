# 클립 납치단 스네이크 — ROADMAP

## 목적
clip-apps 1호. 8비트 레트로 스네이크. **출시 사이클 완주**가 목표(대박 아님).

## 핵심 기능
- 3화면: START → PLAY → GAME OVER → 재시작
- 스와이프 4방향, 먹이 +10·성장, 벽/몸 충돌 게임오버, 최고점수 로컬 저장

## 디자인 원칙
- 8비트 픽셀, 다크배경+네온그린 뱀+빨강 먹이, 픽셀폰트
- 타이틀 "클립 납치단 스네이크" (기념)
- 오프라인. 백엔드·로그인·결제 없음

## 개발 순서
1. 격자 보드+뱀 정적 렌더 + 타이틀   ← 지금
2. 게임 루프+자동 이동
3. 스와이프 입력
4. 먹이+성장+점수
5. 충돌→게임오버
6. 3화면 흐름 연결 + 최고점수 저장
7. 픽셀폰트·색감 폴리시 + 8비트 효과음 + 앱 아이콘
